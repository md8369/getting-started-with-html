UNDERSTANDING THE INTERNET

The internet is a group of computers sending and receiving messages. When a person types a web page it travels across the internet to servers. The servers look up the information requested and sends it back to the browser.

HTML

Hypertext Markup Language is the heart of the web. It contains text content that you can see and markup is how it looks.

TAGS

HTML documents are created using tags. Tags have an opening and a closing depending on which one you are using. Examples of tags are bold,emphasis, anchor and linkable tags to display images or website.
